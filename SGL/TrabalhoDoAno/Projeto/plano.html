Projeto SAP-2

  1. Vamos simular o funcionamento do computador SAP2:


  2. O SAP-2 (Simple As Possible 2) é um modelo educacional de computador criado para ensinar os fundamentos da arquitetura de computadores e montagem de processadores. Ele foi projetado por Albert Paul Malvino no livro "Digital Computer Electronics", sendo uma evolução do SAP-1.
    Além disso, o Sap-2:
    • armazena 65.536 posições de memória;
    • tem uma ALU que soma e subtrai números;
    • 43 instruções;
    • 3 registradores;
    • um display hexadecimal;
    • um acumulador;
    • um sistema de flags;
    • 2 entradas input e uma output;
  3. Vamos colocar a biblioteca p5.js (do Processing) no código para representar o funcionamento (hipoteticamente);
  4. O usuário colocará o código em assembly, gerando o código compilado em hexadecimal e depois ele será executado;
